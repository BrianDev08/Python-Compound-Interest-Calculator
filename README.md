# Python Compound Interest Calculator

A beginner Python project that calculates compound interest while validating user input using `while` loops.

## Features

* Calculates compound interest
* Prevents negative principal amount
* Prevents negative interest rate
* Prevents negative time
* Uses loops until valid input is entered

## Concepts Used

* Variables
* User Input
* float() / int()
* while loops
* if / else
* break
* Mathematical calculations
* f-strings
* pow()

## Formula Used

Final Amount = Principal × (1 + Rate / 100)^Time
